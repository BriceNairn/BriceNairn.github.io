# Brice Nairn Portfolio

Personal portfolio website showcasing my work across geospatial systems, software development, automation and data-driven products.

**Live Site: <a href="https://bricenairn.github.io">https://bricenairn.github.io</a>**

## About

I am a geospatial systems professional with 10+ years of experience across GIS, enterprise spatial platforms, automation and software development. This portfolio highlights a selection of professional and personal projects spanning:

- GIS and spatial systems
- Enterprise ArcGIS platforms
- Python automation
- Data engineering
- Web application development
- Digital product development

## Tech Stack

- [Astro](https://astro.build)
- [tailwindcss](https://tailwindcss.com/)
- [DaisyUI](https://daisyui.com/)
- [GitHub Pages](https://docs.github.com/en/pages)
- [GitHub Actions](https://github.com/features/actions)

## Local Development

```bash
npm install
npm run dev
```

## Deployment

The site is automatically deployed to GitHub Pages using GitHub Actions whenever changes are push to the `main` branch
